# Platform Koşucu (Platform Runner)

Bu proje, Kocaeli Üniversitesi Yazılım Mühendisliği bölümü Programlama dersi kapsamında için geliştirilmiş, C++ ve SFML kütüphanesi kullanan 2 boyutlu bir platform oyunudur.

## 📂 Proje Yapısı
```text
├── src/            # Kaynak kodlar (.cpp ve .hpp dosyaları)
├── assets/         # Görseller, fontlar ve ses efektleri
├── SFML-2.6.2/     # SFML kütüphane bağımlılıkları
├── Frameworks/     # Gerekli framework bileşenleri
├── Makefile        # Derleme otomasyon dosyası
└── README.md       # Proje dokümantasyonu
